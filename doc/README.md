## 📖 文档说明

本项目采用分层架构设计，文档按照功能模块进行分类管理：

1. **需求文档**：记录产品功能需求和业务逻辑
2. **架构文档**：描述系统技术架构和设计思路

4. **SQL文档**：包含数据库相关的脚本和设计文档
5. **部署文档**：指导项目的部署和运维操作
